Tabla en html

 

Las tablas HTML son elementos dentro del diseño web que permiten a los desarrolladores web organizar datos como texto, imágenes y enlaces en filas y columnas.

Formulario en html

Un formulario HTML es una caja de texto que puede contener casillas de verificación, botones y campos alfanuméricos. Suelen encontrarse en los sitios web para recolectar los datos personales de los usuarios interesados en la oferta del sitio. Estos datos posteriormente se envían a una base de datos para ser procesados.

Atributos de table

 

Los atributos de tabla en HTML se definen mediante la etiqueta <table> y tienen algunas etiquetas como <tr>, <td>, <th>, <caption>, <thead>, <tbody>, <tfoot>, <colgroup>, <col> … que se utilizan para completar datos en las tablas de nuestras páginas web. Las tablas permiten a los desarrolladores organizar los datos en filas y columnas.

Etiqueta input

La etiqueta <input> en HTML es un elemento importante para cualquier desarrollador web debido a su versatilidad y su capacidad para recopilar información del usuario, ¡por eso es una herramienta fundamental en la creación de formularios web interactivos!

Etiqueta tr

 

El elemento HTML fila de tabla (table row) <tr> define una fila de celdas en una tabla.

Etiqueta label

El Elemento HTML <label> representa una etiqueta para un elemento en una interfaz de usuario. Este puede estar asociado con un control ya sea mediante la utilizacion del atributo for, o ubicando el control dentro del elemento label. Tal control es llamado "el control etiquetado" del elemento label.

Atributo tr

 

La etiqueta HTML <tr>, abreviatura de "fila de tabla", se utiliza para definir una fila en una tabla HTML . Actúa como contenedor de celdas de datos de tabla (elementos <td>) y también puede incluir celdas de encabezado de tabla (elementos <th>).

Etiqueta select

El elemento select ( <select> ) de HTML representa un control que muestra un menú de opciones. Las opciones contenidas en el menú son representadas por elementos <option> , los cuales pueden ser agrupados por elementos <optgroup> . La opcion puede estar preseleccionada por el usuario.

Etiqueta td y th

 

En HTML, las etiquetas <td> y <th> se utilizan en tablas:

  • <td>: Define una celda de datos en una tabla. Se utiliza para contener el contenido regular de la tabla.
  • <th>: Define una celda de encabezado en una tabla. Se utiliza para encabezados de columnas o filas y generalmente se muestra en negrita y centrada por defecto.

Ambas etiquetas se colocan dentro de la etiqueta <tr> que representa una fila de la tabla.

 

Etiqueta textarea:

La etiqueta <textarea> en HTML se utiliza para crear un área de texto multilinea donde los usuarios pueden ingresar texto. A diferencia de un campo de entrada simple, permite varias líneas de texto.

Atributos de td y th

 

Atributos de <td>:

  • colspan: Número de columnas que abarca.
  • rowspan: Número de filas que abarca.
  • align: Alineación horizontal (left, right, center).
  • valign: Alineación vertical (top, middle, bottom).
  • class: Estilos CSS.
  • style: Estilos en línea.

Atributos de <th>:

  • colspan: Número de columnas que abarca.
  • rowspan: Número de filas que abarca.
  • scope: Alcance del encabezado (col, row).
  • align: Alineación horizontal.
  • valign: Alineación vertical.
  • class: Estilos CSS.
  • style: Estilos en línea.

 

Etiqueta fieldset, legend

Las etiquetas <fieldset> y <legend> se utilizan en HTML para agrupar y describir elementos de formulario:

<fieldset>

  • Descripción: Se utiliza para agrupar controles relacionados dentro de un formulario.
  • Estilo: Generalmente se presenta como un cuadro alrededor de los elementos agrupados.

<legend>

  • Descripción: Proporciona un título o una descripción para el grupo de elementos dentro de un <fieldset>.
  • Uso: Se coloca dentro de un <fieldset> y generalmente se muestra en negrita.

Etiqueta meter

La etiqueta <meter> en HTML se utiliza para mostrar un valor escalar dentro de un rango conocido. Es especialmente útil para representar información como niveles, progresos o resultados cuantitativos.

Etiqueta thead

 

La etiqueta <thead> en HTML se utiliza para agrupar el contenido de los encabezados en una tabla. Se emplea junto con las etiquetas <tbody> y <tfoot> para estructurar mejor la tabla y mejorar la accesibilidad.

Características:

  • Estructura: Permite definir claramente la sección de encabezado de la tabla, lo que ayuda a los lectores de pantalla y otros dispositivos de asistencia a interpretar el contenido.
  • Estilo: Puedes aplicar estilos específicos a los encabezados para diferenciarlos del resto de la tabla.

 

Etiqueta progress

La etiqueta <progress> en HTML se utiliza para mostrar el progreso de una tarea en curso. Es especialmente útil para indicar cómo avanza una operación, como la descarga de un archivo o la carga de un recurso.

Etiqueta tfoot

 

La etiqueta <tfoot> se utiliza en HTML para definir un pie de tabla. Se coloca dentro de una tabla y generalmente contiene información resumen, totales o cualquier otra información relevante que se desee presentar al final de la tabla. Es importante que la etiqueta <tfoot> se coloque antes de <tbody> en el código para que los navegadores la interpreten correctamente.

 

Etiqueta datalist

La etiqueta <datalist> se utiliza en HTML para proporcionar una lista de opciones predefinidas que se pueden seleccionar en un campo de entrada (<input>). Esto es útil para sugerir opciones a los usuarios mientras escriben, mejorando la experiencia de usuario al facilitar la entrada de datos.

Etiqueta tbody

 

La etiqueta <tbody> se utiliza en HTML para agrupar el contenido del cuerpo de una tabla. Se coloca entre las etiquetas <thead> (que define el encabezado de la tabla) y <tfoot> (que define el pie de la tabla, si se utiliza). Esta etiqueta ayuda a estructurar mejor la tabla y puede facilitar el estilo y la manipulación con JavaScript.

Etiqueta summary

La etiqueta <summary> se utiliza en HTML para crear un resumen o título que se puede expandir o contraer en un elemento <details>. Esta combinación permite mostrar información adicional de manera opcional, mejorando la usabilidad y la presentación de contenido en una página web.

 

Tema 2. Define los elementos que componen la estructuración de una página web:

  1. Contenedor (wrapper)
  2. Cabecera (header)
  3. Contenido (content
  4. Menú (menu)
  5. Pie (footer)
  6. Lateral (sidebar)
  7. Etiqueta div

 

1. Contenedor (wrapper)

El contenedor, o "wrapper", es un elemento HTML que se utiliza para envolver el contenido de una página web. Su propósito principal es proporcionar un marco o estructura para el diseño. A menudo se utiliza para controlar el ancho total de la página y centrar el contenido en la pantalla.

2. Cabecera (header)

La cabecera es la parte superior de la página web y suele contener el logotipo, el título del sitio y, a veces, el menú de navegación. Se define mediante la etiqueta <header> y establece el contexto y la identidad de la página.

3. Contenido (content)

El contenido es la parte principal de la página que incluye el texto, imágenes, videos y cualquier otro tipo de información que el usuario necesita ver. Se puede definir utilizando etiquetas como <main> o simplemente divs (<div>), y es el área donde se desarrolla la interacción principal del usuario.

4. Menú (menu)

El menú es una lista de enlaces que permite la navegación entre diferentes secciones o páginas del sitio web. Se suele implementar utilizando listas desordenadas (<ul> y <li>) y puede estar ubicado en la cabecera o en un lateral. En HTML5, se puede usar la etiqueta <nav> para definir una sección de navegación.

5. Pie (footer)

El pie de página, o "footer", es la sección situada al final de la página. Generalmente incluye información como derechos de autor, enlaces a políticas de privacidad, información de contacto y otros recursos importantes. Se define mediante la etiqueta <footer>.

6. Lateral (sidebar)

La barra lateral, o "sidebar", es un área que se coloca generalmente a un lado del contenido principal. Puede contener elementos como menús de navegación secundarios, widgets, anuncios o enlaces relacionados. A menudo se implementa con divs (<div>), y puede ser estática o dinámica.

7. Etiqueta div

La etiqueta <div> es un contenedor genérico en HTML que se utiliza para agrupar contenido y aplicar estilos. No tiene un significado semántico específico, pero es muy útil para estructurar el diseño y aplicar CSS. Se usa a menudo para dividir secciones de la página, facilitando la organización y el diseño responsivo.

 

Tema 3. Identifica y describe los elementos que componen la estructuración semántica de una página web.

 

La estructuración semántica de una página web se refiere al uso de etiquetas HTML que describen el contenido de manera significativa, mejorando la accesibilidad y el SEO. Aquí están los elementos clave que componen esta estructuración:

1. <header>

La etiqueta <header> se utiliza para definir la cabecera de un documento o de una sección. Generalmente incluye el logotipo, el título del sitio y el menú de navegación. Proporciona contexto sobre el contenido que sigue.

2. <nav>

La etiqueta <nav> se utiliza para encapsular la navegación principal de la página. Contiene enlaces a las secciones más importantes del sitio, facilitando la navegación para los usuarios y los motores de búsqueda.

3. <main>

La etiqueta <main> representa el contenido principal del documento. Debe contener información relevante y única, excluyendo elementos como encabezados, pies de página y menús. Es crucial para ayudar a los lectores de pantalla a identificar el contenido más importante.

4. <article>

La etiqueta <article> se utiliza para definir contenido independiente que puede ser distribuido o reutilizado, como entradas de blog, noticias o publicaciones. Cada artículo debe ser comprensible por sí mismo, incluso fuera del contexto de la página.

5. <section>

La etiqueta <section> se usa para agrupar contenido temáticamente relacionado. Cada sección puede tener un encabezado (<h1>, <h2>, etc.) y proporciona una estructura jerárquica a la información.

6. <aside>

La etiqueta <aside> se utiliza para contenido relacionado pero no esencial al contenido principal, como notas al margen, enlaces adicionales o publicidad. Proporciona contexto adicional sin interrumpir la narrativa principal.

7. <footer>

La etiqueta <footer> se utiliza para definir el pie de página de un documento o sección. Generalmente incluye información de contacto, derechos de autor, enlaces a políticas y otros recursos importantes.

8. <h1> a <h6>

Las etiquetas de encabezado (<h1> a <h6>) se utilizan para definir la jerarquía del contenido. <h1> es el encabezado principal y debe usarse una sola vez por página, mientras que las demás etiquetas se utilizan para subsecciones.

9. <figure> y <figcaption>

La etiqueta <figure> se usa para encapsular contenido visual, como imágenes o gráficos, y se puede acompañar de una etiqueta <figcaption> para proporcionar una descripción o título.

10. <time>

La etiqueta <time> se utiliza para representar una fecha o un rango de tiempo, lo que ayuda a los motores de búsqueda a entender mejor el contenido relacionado con el tiempo.
